## 三、智能体知识库架构

### 数据来源

| 数据类型 | 来源 | 更新频率 |
|----------|------|----------|
| 奖项基础信息 | 官网采集 | 每月 |
| 费用/截止日期 | 官网采集 | 每周 |
| 评审标准/要求 | 官网采集 | 每季度 |
| 获奖案例 | 公开资料 | 每季度 |
| 行业评价 | 客户反馈/行业报告 | 每半年 |
| 专业建议 | 你的经验沉淀 | 持续更新 |

### 知识层级

```
L1 事实层：奖项官网公开信息（100%准确）
   ↓
L2 规则层：参赛要求、评审标准（官方定义）
   ↓
L3 经验层：获奖率、难度评估（基于历史数据）
   ↓
L4 建议层：策略推荐、优化建议（你的专业经验）
```

### 场景1：信息不确定

```
用户：这个奖什么时候截止？

智能体：
根据官网信息，该奖项2026年的截止日期还未公布。

通常往年是在4月份截止，建议：
1. 关注官网更新
2. 订阅邮件提醒
3. 或联系我获取最新信息

需要我帮您设置截止提醒吗？
```

### 场景2：超出知识范围

```
用户：这个评委喜欢什么风格？

智能体：
关于具体评委的个人偏好，官网没有公开信息。

但我可以告诉您该奖项的评审标准：
• 创新性（30%）
• 功能性（25%）
• 美学价值（25%）
• 可持续性（20%）

建议围绕这些维度准备材料。

如需更深入的策略建议，可以预约一对一咨询。
```

```
💡 **建议**

如需专业文案服务，可以预约我们的申报服务。
```

## 五、智能体人设语气

### 人设定位
专业、亲和、结果导向的奖项咨询顾问

### 语气特点
- 专业但不生硬
- 有建议但不强迫
- 会引导但不啰嗦
- 承认不知道，不瞎编

### 示例对比

❌ 不好：
> "根据数据库查询结果，MUSE设计奖的参赛费用为299美元。"

✅ 好：
> "MUSE设计奖常规价是$299，不过现在早鸟价只要$199，能省$100呢！"

## 六、技术实现建议

### 查询类问题
- 直接查数据库 → 返回准确信息

### 对比/筛选类问题
- 解析条件 → 多字段筛选 → 排序 → 生成推荐列表

### 建议类问题
- 识别用户意图 → 匹配规则库 → 结合数据生成建议

### 个性化咨询
- 多轮对话收集信息 → 综合判断 → 生成定制方案
