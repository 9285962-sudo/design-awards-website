# AI辅助采集方案对比分析

---

## 方案对比

| 维度 | 方案A：AI自主采集（主动爬虫） | 方案B：AI辅助填写（你给模板） |
|------|------------------------------|------------------------------|
| **工作流程** | 你提供奖项官网URL → AI自动访问 → 提取信息 → 返回结构化数据 | 你访问官网 → 复制网页内容 → 粘贴给AI → AI按模板提取 → 返回结构化数据 |
| **你的操作** | 只需提供URL列表 | 需要访问每个官网、复制内容 |
| **AI工作** | 访问网页、解析HTML、提取数据 | 解析你提供的文本、按模板提取 |
| **技术门槛** | 需要爬虫技术/自动化工具 | 只需要ChatGPT/Claude等AI工具 |
| **成本** | 服务器+代理IP+开发成本 | AI API调用费（较低） |
| **准确性** | 依赖网页结构稳定性 | 依赖你复制的内容完整性 |
| **速度** | 批量自动化，100个奖项几分钟 | 逐个处理，100个奖项需要几小时 |
| **灵活性** | 网站改版需要重新配置 | 随时可调整，不受网站改版影响 |

## 方案B详细流程（推荐现阶段）

### 工作流程

```
你访问奖项官网
    ↓
复制关键页面内容（或截图）
    ↓
粘贴到AI对话框
    ↓
AI按模板提取20个字段
    ↓
返回标准化JSON/表格
    ↓
你审核并录入数据库
```

### 给AI的Prompt模板

```
你是一个专业的奖项信息提取助手。请从以下网页内容中提取奖项信息，按指定格式返回。

## 提取字段（共20个）：

1. 奖项中文名(award_name_cn)
2. 奖项英文名(award_name_en)
3. 主办机构(organizer)
4. 主办国家(country)
5. 官网链接(website)
6. 主领域(category_main)：建筑/室内/产品/平面/UIUX
7. 细分类别(category_sub)：数组格式
8. 参赛费用(entry_fee)：数字，单位USD
9. 早鸟截止日期(deadline_early)：YYYY-MM-DD格式
10. 常规截止日期(deadline_regular)：YYYY-MM-DD格式
11. 最终截止日期(deadline_final)：YYYY-MM-DD格式
12. 作品年限要求(project_year_limit)：如"2年内"
13. 项目状态要求(project_status)：概念/建成/不限
14. 评审标准(judging_criteria)：文本摘要
15. 评审周期(judging_cycle)：如"3个月"
16. 结果公布时间(announcement_date)：YYYY-MM-DD格式
17. 获奖难度(difficulty_level)：高/中/低
18. 知名度等级(prestige_level)：顶级/知名/新锐
19. 获奖权益(winner_benefits)：文本摘要
20. 特殊说明(special_notes)：其他重要信息

## 网页内容：

[粘贴网页内容]

## 输出格式：
请以JSON格式返回，字段名用英文，值为提取的内容。如果某字段未找到，值设为null。
```

## 两种方案的适用场景

### 选择方案A（AI自主采集）当：
- ✅ 你有技术团队或能外包开发
- ✅ 奖项数量 > 50个，且更新频繁
- ✅ 需要实时/定期自动更新
- ✅ 愿意投入初期开发成本

### 选择方案B（AI辅助填写）当：
- ✅ 你目前一个人操作
- ✅ 奖项数量 < 50个，或更新不频繁
- ✅ 想快速启动，不想等开发
- ✅ 愿意投入人工复制粘贴的时间

---

## 混合方案（最佳实践）

### 阶段1：AI辅助填写（现在）

**操作步骤**：
1. 准备20个核心奖项列表
2. 逐个访问官网，复制关键页面内容
3. 用AI批量提取（一次可以处理3-5个奖项）
4. 审核结果，录入Excel/数据库

**时间估算**：
- 20个奖项 × 5分钟 = 100分钟（首次录入）
- 每月更新：20个奖项 × 2分钟 = 40分钟

### 阶段2：半自动化（3个月后）

当数据量大了之后：
- 用简单爬虫采集5-10个高频奖项
- 其他继续用AI辅助填写
- 建立变更提醒机制

### 阶段3：全自动化（6个月后）

当业务验证成功：
- 投入开发配置化爬虫
- AI作为兜底方案
- 实现全自动更新

## 实操建议

### 立即可用的AI工具

| 工具 | 适用场景 | 成本 |
|------|----------|------|
| ChatGPT-4 | 复杂页面解析、多字段提取 | $20/月 |
| Claude 3 | 长文本处理、表格生成 | $20/月 |
| 讯飞星火/文心一言 | 中文奖项信息提取 | 免费/低价 |

### 优化技巧

1. **批量处理**
   ```
   请同时处理以下3个奖项的信息提取：
   
   【奖项1：MUSE Design Awards】
   [内容]
   
   【奖项2：TITAN Property Awards】
   [内容]
   
   【奖项3：IDA Design Awards】
   [内容]
   ```

2. **提供示例**
   ```
   参考以下示例格式：
   {
     "award_name_cn": "缪斯设计奖",
     "award_name_en": "MUSE Design Awards",
     "entry_fee": 299,
     "deadline_regular": "2026-03-15"
   }
   ```

3. **迭代优化**
   - 第一次提取后检查错误
   - 告诉AI哪里错了，让它学习
   - 保存优化后的Prompt作为模板

## 总结

| 你的情况 | 推荐方案 |
|----------|----------|
| 想快速启动、验证MVP | **方案B：AI辅助填写** |
| 有开发资源、追求自动化 | **方案A：AI自主采集** |
| 长期运营、数据量大 | **混合方案** |

**现阶段建议**：用AI辅助填写快速启动，积累数据的同时验证小程序价值，等业务跑通后再考虑自动化。
