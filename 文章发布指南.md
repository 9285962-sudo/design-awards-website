# 文章发布指南

## 文件夹结构（桌面快捷方式）

```
桌面
├── 🟢 赛事动态.lnk
│   └── content/赛事动态/
│       ├── 报名通知/     ← 报名开始、截止提醒
│       ├── 获奖快讯/     ← 获奖通报、获奖作品
│       └── 赛事动态/     ← 奖项动态、行业新闻
│
└── 🔵 参赛策略.lnk
    └── content/参赛策略/
        ├── 参赛策略/     ← 参赛技巧、注意事项
        ├── 荣誉转化/     ← 获奖后如何转化价值
        └── 参赛成长/     ← 通过参赛提升设计能力
```

## 发布步骤

### 1. 写文章

**文件命名**：`YYYY-MM-DD-标题.md`

**Front Matter**（非必需）：
```yaml
---
title: 文章标题
date: YYYY-MM-DD
author: 设计能
summary: 一句话概括（用于列表页显示）
---
```

**封面图**：放在文章最前面（第一张图），自动作为封面。

### 2. 放文章

根据文章内容，放到对应的**三级文件夹**：

| 快捷方式 | 子文件夹 | 内容 |
|---------|---------|------|
| 赛事动态 | 报名通知 | 报名开始、截止提醒 |
| 赛事动态 | 获奖快讯 | 获奖通报、获奖作品 |
| 赛事动态 | 赛事动态 | 奖项动态、行业新闻 |
| 参赛策略 | 参赛策略 | 参赛技巧、注意事项 |
| 参赛策略 | 荣誉转化 | 获奖后如何转化价值 |
| 参赛策略 | 参赛成长 | 通过参赛提升设计能力 |

### 3. 一键发布

双击桌面 `设计能网站一键发布.bat`

自动完成：
1. 扫描所有 .md 文件
2. 构建文章数据
3. 构建网站
4. 部署到 Cloudflare

### 4. 验证

访问：https://www.52de.cc/news/

## 注意事项

- **不需要写 category**，构建时自动用文件夹名
- **封面图放最前面**，会自动提取为封面
- **正文配图**按正常顺序写即可
