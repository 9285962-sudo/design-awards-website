# 国际设计大奖网站部署指南

## 第一步：创建GitHub仓库（2分钟）

1. 打开 https://github.com/new
2. **Repository name** 填写：`design-awards-website`
3. **Description** 填写：`国际设计大奖参赛咨询网站`
4. 选择 **Public**（公开仓库免费）
5. 勾选 **Add a README file**（可选）
6. 点击 **Create repository**

## 第二步：运行推送脚本（1分钟）

1. 打开文件夹：`C:\Users\Administrator\WorkBuddy\国际设计大奖参赛咨询`
2. 双击运行 `setup-github.bat`
3. 按提示输入你的GitHub用户名
4. 等待推送完成

## 第三步：部署到Vercel（2分钟）

1. 打开 https://vercel.com
2. 点击 **Sign Up**
3. 选择 **Continue with GitHub**（用GitHub账号登录）
4. 点击 **Add New Project**
5. 在列表中找到 `design-awards-website`
6. 点击 **Import**
7. 点击 **Deploy**

等待部署完成，网站就上线了！

## 第四步：配置自动更新（1分钟）

1. 在Vercel项目页面，点击 **Settings**
2. 点击 **Git**
3. 确认 **Production Branch** 是 `main`
4. 完成！

## 验证部署

部署完成后，Vercel会给你分配一个域名，例如：
```
https://design-awards-website.vercel.app
```

打开这个链接，就能看到你的网站了！

## 后续使用

配置完成后，你只需要：

1. **添加文章URL** 到 `scripts/image-crawler/urls/pending-articles.json`
2. **保存文件**

系统会自动：
- 每天凌晨2点采集新文章
- 自动生成文章页面
- 自动部署更新

你什么都不用做！

## 自定义域名（可选）

如果你想用自己的域名：

1. 在Vercel项目页面点击 **Settings** -> **Domains**
2. 输入你的域名，例如 `www.yourdomain.com`
3. 按提示添加DNS记录
4. 等待DNS生效（通常几分钟到几小时）

## 常见问题

### Q: 推送失败怎么办？
A: 检查：
- GitHub仓库是否已创建
- 仓库名是否为 `design-awards-website`
- 网络连接是否正常

### Q: 部署失败怎么办？
A: 检查：
- 代码是否已推送到GitHub
- Vercel是否正确导入了项目

### Q: 如何更新网站内容？
A: 只需编辑 `pending-articles.json` 添加新文章URL，系统会自动处理。
