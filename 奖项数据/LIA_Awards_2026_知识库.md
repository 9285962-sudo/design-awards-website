## 作品要求

### 视频格式 (MOV & MP4)

| 参数 | 要求 |
|------|------|
| 标清最低分辨率 | 720 x 480 |
| 高清分辨率 | 1920 x 1080 |
| 帧率 | 原始帧率 |
| 编码 | H264 或 Mpeg4 |
| 码率 | 5,000-10,000kbps |
| 音频 | AAC 48KHz |
| 最大文件 | 1GB |

**禁止**: 视频中不得出现Slate、Clock、Bars，不得包含参赛者名称或公司品牌标识

### 图片格式 (JPEG)

| 参数 | 要求 |
|------|------|
| 分辨率 | 300 dpi |
| 色彩模式 | RGB |
| 最大文件 | 6MB |
| 竖版 | 约 2400 x 3000 像素 |
| 横版 | 约 3000 x 2400 像素 |

### 音频格式 (MP3)

| 参数 | 要求 |
|------|------|
| 码率 | 64-196kbps |
| 类型 | MP3 (立体声) |
| 最大文件 | 3MB |

### 上传限制

- **大多数类别**: 最多3个文件
- **单媒体类别**: 最多1个文件 (Music & Sound, Music Video, Online Film, Production & Post, TV & Cinema)

### 案例研究要求

以下类别**必须提交案例研究**:
- Business Transformation
- Creative Strategy
- Creative Use of Data
- Creativity In Media / PR
- Cultural Catalyst
- Democracy and Human Rights
- Direct
- Entertainment & Content
- Evolution
- Gaming
- Non-Traditional

**案例研究规格**:
- 时长: 不超过3分钟
- 语言: 英文或带英文字幕
- 音乐: 必须使用免版税音乐
- 匿名: 不得包含公司名称或个人署名

## 重要规则

### AI使用规定
- **创意作品**: 必须合乎伦理，明确声明AI使用情况，有人类监督
- **案例片**: 禁止使用AI操纵图像或重建不准确数据

### 音乐要求
- 必须使用免版税音乐或已授权音乐
- 参赛者需承担音乐版权相关责任

### 违规处罚
- 如发现作品或案例片被操纵，奖项将被公开撤回
- 参赛者将被取消资格且不退款
