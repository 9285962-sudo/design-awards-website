# 国际设计大奖智能助手 - 小程序

## 项目结构

```
mini-program/
├── app.js                 # 全局应用逻辑
├── app.json               # 全局配置
├── app.wxss               # 全局样式
├── project.config.json    # 项目配置
├── sitemap.json           # 搜索索引
└── pages/
    ├── index/             # 首页
    │   ├── index.wxml
    │   ├── index.wxss
    │   ├── index.js
    │   └── index.json
    ├── chat/              # 智能咨询页
    │   ├── chat.wxml
    │   ├── chat.wxss
    │   ├── chat.js
    │   └── chat.json
    ├── award-detail/      # 奖项详情页
    │   ├── award-detail.wxml
    │   ├── award-detail.wxss
    │   ├── award-detail.js
    │   └── award-detail.json
    └── profile/           # 个人中心
        ├── profile.wxml
        ├── profile.wxss
        ├── profile.js
        └── profile.json
```

## 功能模块

### 1. 首页 (index)
- 欢迎语和数据统计展示
- 快速入口（智能咨询、浏览奖项）
- 热门奖项推荐（MUSE奖卡片）
- 服务介绍
- CTA引导

### 2. 智能咨询 (chat)
- 对话式交互界面
- 快捷问题推荐
- 基于MUSE奖数据的智能回复
- 支持多种回答类型（文本、卡片、列表）

**支持的咨询类型**：
- 费用查询
- 截止日期
- 获奖率/难度
- 参赛类别
- 作品要求/规格
- 新手指导
- 获奖权益

### 3. 奖项详情 (award-detail)
- 奖项基本信息
- 参赛费用时间线
- 关键时间节点
- 参赛类别
- 作品要求
- 作品规格（图片/视频）
- 评审标准
- 奖项等级
- 获奖权益
- 专家建议

### 4. 个人中心 (profile)
- 用户信息展示
- 数据统计
- 我的服务（收藏、历史、提醒、进度）
- 专家服务（咨询、套餐、案例）
- 关于我们

## 数据说明

### MUSE设计奖数据
已内置完整的MUSE设计奖数据，包括：
- 基础信息（名称、主办、国家）
- 费用信息（5个阶段）
- 时间节点（投稿、评审、结果、典礼）
- 参赛类别（7大类，450+细分）
- 作品要求（年限、状态、语言）
- 作品规格（图片、视频、文档）
- 评审标准（5维度）
- 奖项等级（铂金/金/银/玫瑰金）
- 获奖权益（10项）
- 专家评价（难度、费用、策略）

## 使用方式

### 开发者工具运行
1. 打开微信开发者工具
2. 导入项目，选择 `mini-program` 文件夹
3. 使用测试号或自己的AppID
4. 点击编译运行

### 测试对话功能
在咨询页面，可以尝试提问：
- "MUSE奖费用多少？"
- "什么时候截止？"
- "获奖率怎么样？"
- "需要什么材料？"
- "推荐个建筑类的奖"
- "新手适合投哪个？"
