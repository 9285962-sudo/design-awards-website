# 智能体问答修复记录

## 2026-04-05 修复内容

### 1. 退款政策回复优化
**问题**："MUSE奖没获奖能退费吗？"答非所问
**原因**：数据文件中 `refund_policy` 字段值为"不退费"，直接显示过于简短
**修复**：使用完整描述代替字段值

### 2. 问题匹配关键词扩展

| 问题类型 | 新增匹配关键词 |
|---------|---------------|
| 类别 | "哪些" + "参赛" |
| 概念设计 | "接受" + "概念" |
| 图片规格 | "要求" + "多少" |
| 语言 | "接受" + "中文" |
| 评审标准 | "评审" + "是什么" |

| 材料要求 | "有什么" + "要求" / "需要" + "什么" |
