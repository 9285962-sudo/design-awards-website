# MUSE 历届获奖信息采集 - 手动执行指南

## 前置要求

确保已安装 Node.js (v16+)

## 执行步骤

### 步骤1: 打开终端
按 `Win + R`，输入 `cmd` 回车，打开命令提示符

### 步骤2: 进入项目目录
```bash
cd "C:\Users\Administrator\WorkBuddy\国际设计大奖参赛咨询"
```

### 步骤3: 安装 Playwright
```bash
npm install playwright
```

### 步骤4: 安装浏览器
```bash
npx playwright install chromium
```

### 步骤5: 运行采集脚本
```bash
node scripts/crawl-muse-winners.js
```

## 预期结果

脚本会自动：
1. 打开 Chrome 浏览器
2. 访问 MUSE 获奖者画廊页面
3. 采集 2021-2025 年的获奖项目
4. 保存数据到 `奖项数据/MUSE历届获奖数据/` 目录

## 采集数据格式

每个获奖项目包含：
```json
{
  "title": "项目名称",
  "designer": "设计师/公司",
  "category": "设计类别",
  "awardLevel": "获奖等级 (Platinum/Gold/Silver)",
  "year": 2025,
  "imageUrl": "作品图片链接",
  "projectUrl": "项目详情页链接"
}
```

## 注意事项

- 采集过程会打开浏览器窗口，请勿关闭
- 每个年份采集后会自动保存
- 如遇反爬，会添加随机延迟
- 完整采集预计需要 5-10 分钟

## 备选方案

如果官网有反爬机制，可以尝试：
1. 降低采集频率（修改脚本中的 delay 时间）
2. 使用代理（在脚本中配置 proxy）
3. 分多次采集不同年份

## 故障排除

**问题1: 提示找不到 playwright**
```bash
npm list playwright
# 如果没安装，执行：
npm install playwright
```

**问题2: 提示浏览器未安装**
```bash
npx playwright install chromium
```

**问题3: 页面加载超时**
- 检查网络连接
- 修改脚本中的 waitUntil 参数为 'domcontentloaded'
- 增加 delay 时间
