# 国际设计大奖智能体小程序 - 设计方案

---

## 一、产品定位

**产品名称**：设计奖项通 / 奖项智选（待定）

**核心定位**：专业的设计大奖智能咨询助手，展示专业性的同时，通过分层信息策略保护商业壁垒。

**目标用户**：
- 主要：设计事务所设计师（潜在客户）
- 次要：你自己（作为专业工具展示）

---

## 二、核心策略：信息分层设计

### 三层信息架构

| 层级 | 信息类型 | 访问权限 | 目的 |
|------|----------|----------|------|
| **L1 公开层** | 奖项名称、Logo、主办国家、领域分类、简介 | 无需登录 | 展示专业性，建立信任 |
| **L2 注册层** | 参赛时间、费用、类别细分、评审标准、往届获奖作品 | 微信登录+手机号 | 获取潜在客户线索 |
| **L3 专属层** | 获奖概率测评、申报策略建议、一对一咨询入口 | 人工审核/付费 | 核心商业服务转化 |

---

## 三、功能模块设计

### 模块1：奖项探索（公开层）

**功能**：
- 奖项列表浏览（卡片式）
- 基础筛选：领域（建筑/室内/产品/平面/UIUX）、地区（美洲/欧洲/亚太）
- 奖项搜索
- 奖项详情页（基础信息）

**展示内容**：
- 奖项Logo
- 奖项名称（中/英文）
- 主办国家/机构
- 适用领域标签
- 一句话简介
- 【登录查看详情】按钮

---

### 模块2：智能筛选（注册层）

**触发条件**：用户点击"深度筛选"或"查看详情"

**弹窗引导**：
> "解锁完整奖项信息，获取个性化推荐
> 请登录并授权手机号"

**筛选维度**（20个字段中的关键筛选项）：

| 筛选类型 | 选项示例 |
|----------|----------|
| 设计领域 | 建筑设计、室内设计、产品设计、平面设计、UI/UX |
| 项目类型 | 概念设计、已建成/已上市、改造翻新 |
| 预算区间 | 免费、¥1000以下、¥1000-3000、¥3000-5000、¥5000+ |
| 截止时间 | 本月截止、3个月内、半年内、全年可投 |
| 知名度 | 顶级（红点/iF/IDEA）、知名、新锐 |
| 获奖难度 | 高竞争、中等、相对容易 |
| 地域偏好 | 欧洲奖项、美洲奖项、亚太奖项、不限 |
| 作品年限 | 限2年内、限3年内、限5年内、不限 |

**筛选结果**：
- 符合条件的奖项列表
- 支持多奖项对比（最多3个）
- 每个奖项显示匹配度评分

---

### 模块3：奖项对比（注册层）

**功能**：选中2-3个奖项进行并列对比

**对比维度**：
- 参赛费用
- 截止时间
- 评审周期
- 获奖权益
- 往届获奖率
- 适合项目类型

**行动引导**：
> "不确定选哪个？预约免费咨询，获取专业建议"

---

### 模块4：我的收藏（注册层）

- 收藏感兴趣的奖项
- 设置参赛提醒
- 查看历史浏览

---

### 模块5：智能测评（L3专属层 - 未来扩展）

**获奖概率测评**：
- 上传项目基本信息
- 回答5-8个关键问题
- AI评估获奖概率（高/中/低）
- 推荐最适合的3个奖项

**获奖作品分析**：
- 上传往届获奖作品
- AI分析获奖理由
- 提炼可借鉴的申报策略

**入口控制**：
- 显示"限时免费体验"或"咨询顾问解锁"
- 引导添加微信/预约咨询

---

### 模块6：专家咨询（转化入口）

**展示内容**：
- 你的专业背景（人设定位）
- 90%获奖率数据
- 服务200+设计事务所案例
- 客户评价（精选）

**行动按钮**：
- 【预约免费咨询】
- 【获取申报方案】
- 【加入服务群】

---

## 四、用户权限与访问控制

### 权限层级

```
游客（未登录）
  └── 可浏览奖项列表、基础信息
  └── 可使用基础筛选（领域、地区）
  └── 不可查看：费用、截止日期、详细评审标准

注册用户（微信登录+手机号授权）
  └── 解锁全部奖项信息
  └── 使用高级筛选
  └── 收藏奖项、设置提醒
  └── 使用对比功能
  └── 查看部分AI测评（限次数）

VIP客户（人工审核/付费）
  └── 无限次AI测评
  └── 一对一咨询优先权
  └── 定制化申报方案
```

### 登录流程设计

1. 用户点击"查看详情"或"深度筛选"
2. 弹出登录引导页：
   - 标题："解锁专业奖项信息"
   - 卖点：100+国际奖项、20+筛选维度、90%客户获奖率
   - 按钮："微信一键登录"
3. 微信授权后，请求手机号授权
4. 登录成功，自动返回之前页面

## 六、页面结构与交互流程

### 页面地图

```
首页
  ├── 奖项探索
  │     ├── 奖项列表
  │     ├── 奖项详情（基础版）
  │     └── 登录引导弹窗
  ├── 智能筛选（需登录）
  │     ├── 筛选条件页
  │     ├── 筛选结果
  │     └── 奖项对比
  ├── 我的（需登录）
  │     ├── 我的收藏
  │     ├── 参赛提醒
  │     └── 浏览历史
  └── 专家咨询
        ├── 顾问介绍
        ├── 服务介绍
        └── 预约入口
```

### 核心流程

**流程1：浏览→注册→筛选**
```
用户进入首页 → 浏览奖项卡片 → 点击"查看详情" 
→ 弹出登录引导 → 微信登录+手机号授权 
→ 进入奖项详情（完整版）→ 使用智能筛选
→ 收藏/对比/预约咨询
```

**流程2：直接筛选**
```
用户点击"智能筛选" → 弹出登录引导 
→ 登录后进入筛选页 → 选择条件 
→ 查看结果 → 对比奖项 → 预约咨询
```
