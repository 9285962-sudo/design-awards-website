# 奖项数据采集方案 - 全量版（含兜底知识库）

## 一、采集架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                      用户提问                                │
└──────────────────────┬──────────────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  第一层：结构化字段（快速响应80%高频问题）                      │
│  • 费用、截止、国家、主办方、获奖率...                         │
│  • 毫秒级查询，精准回答                                       │
└──────────────────────┬──────────────────────────────────────┘
                       ▼ 字段无法回答
┌─────────────────────────────────────────────────────────────┐
│  第二层：全量知识库（兜底响应20%长尾问题）                      │
│  • 完整官网内容、FAQ、评审团介绍                              │
│  • 历届获奖信息（设计师、作品、年份、等级）                    │
│  • RAG检索 + 大模型生成回答                                  │
└─────────────────────────────────────────────────────────────┘
```

## 三、第二层：全量知识库（新增）

### 3.1 官网全量内容采集

#### A. 静态页面内容
| 页面 | 内容 | 用途 |
|------|------|------|
| About/关于我们 | 奖项历史、使命、价值观 | 回答"这个奖有什么背景" |
| FAQ/常见问题 | 所有Q&A | 回答各种细节问题 |
| Judging/评审 | 评审团介绍、评审流程 | 回答"谁评审""怎么评" |
| Terms/条款 | 参赛条款、隐私政策 | 回答法律相关问题 |
| Press/新闻 | 媒体报道、新闻稿 | 回答知名度相关问题 |

#### B. 动态内容采集
| 内容类型 | 采集方式 | 更新频率 |
|----------|----------|----------|
| 公告/通知 | 官网公告栏 | 每日检查 |
| 博客文章 | 设计趋势、获奖故事 | 每周检查 |
| 社交媒体 | Instagram/Facebook内容 | 每周同步 |


#### 采集范围
| 数据维度 | 说明 | 价值 |
|----------|------|------|
| 近5年获奖作品 | 2021-2025赛季 | 了解最新趋势 |
| 铂金奖作品 | 最高荣誉作品 | 标杆参考 |
| 中国设计师获奖 | 国内同行案例 | 增强信任感 |
| 各类别获奖分布 | 哪个类别竞争激烈 | 选类策略参考 |

#### 采集来源
1. **官网获奖者画廊** - 基础信息
2. **官方年鉴** - 完整信息（如有）
3. **设计师官网** - 补充信息
4. **设计媒体报道** - 获奖故事

## 五、知识库应用方式

### 5.1 查询流程
```
用户提问
    ↓
[意图识别] → 需要哪个奖项的什么信息？
    ↓
[第一层查询] → 结构化字段中有吗？
    ↓ 有 → 直接返回精准答案
    ↓ 无
[第二层检索] → RAG检索全量知识库
    ↓
[大模型生成] → 基于检索内容生成回答
    ↓
返回答案 + 引用来源
```

### 5.2 回答示例

**用户问**："MUSE奖的评审团都有谁？"

**第一层查询**：结构化字段中没有评审团详细信息

**第二层检索**：从全量知识库中找到评审团介绍页面

**生成回答**：
```
🏆 MUSE设计奖评审团

MUSE设计奖由国际奖项协会（IAA）评审委员会组成，包括：
• 全球知名创意总监
• 资深设计师
• 行业执行制作人

评审维度：
• 创意与原创性（30%）
• 设计理念与策略（25%）
• 功能性与实用性（20%）
• 美学与视觉效果（15%）
• 社会影响力与可持续性（10%）

📌 来源：MUSE官网评审标准页面
```
