## 字段分类说明

### 基础信息（5个字段）
| 字段 | 用途 |
|------|------|
| award_name_cn | 中文展示 |
| award_name_en | 英文展示、搜索 |
| organizer | 了解主办方背景 |
| country | 地域筛选 |
| website | 跳转官网 |

### 分类信息（2个字段）
| 字段 | 用途 |
|------|------|
| category_main | 一级筛选（建筑/室内/产品等） |
| category_sub | 二级筛选（住宅/商业/概念等） |

### 参赛信息（5个字段）
| 字段 | 用途 |
|------|------|
| entry_fee | 预算筛选 |
| deadline_early | 早鸟策略 |
| deadline_regular | 常规参赛 |
| deadline_final | 延期参赛 |
| project_year_limit | 作品时效性筛选 |

### 项目要求（1个字段）
| 字段 | 用途 |
|------|------|
| project_status | 判断项目是否符合参赛条件 |

### 评审信息（3个字段）
| 字段 | 用途 |
|------|------|
| judging_criteria | 了解评审侧重点 |
| judging_cycle | 评估等待时间 |
| announcement_date | 规划获奖后宣传 |

### 奖项属性（2个字段）
| 字段 | 用途 |
|------|------|
| difficulty_level | 评估获奖概率 |
| prestige_level | 评估奖项含金量 |

### 获奖权益（1个字段）
| 字段 | 用途 |
|------|------|
| winner_benefits | 了解获奖价值 |

### 其他（1个字段）
| 字段 | 用途 |
|------|------|
| special_notes | 记录特殊规则、注意事项 |

## 枚举值定义

### category_main（主领域）
```
- 建筑设计 (architecture)
- 室内设计 (interior)
- 产品设计 (product)
- 平面设计 (graphic)
- UIUX设计 (uiux)
- 综合设计 (comprehensive)
```

### project_status（项目状态）
```
- 概念设计 (concept)
- 已建成/已上市 (built)
- 不限 (any)
```

### difficulty_level（获奖难度）
```
- 高 (high) - 获奖率<10%
- 中 (medium) - 获奖率10-30%
- 低 (low) - 获奖率>30%
```

### prestige_level（知名度等级）
```
- 顶级 (premium) - 红点、iF、IDEA等
- 知名 (established) - 行业公认的老牌奖项
- 新锐 (emerging) - 新兴但有潜力的奖项
```
